# Terminology Guidance

Use this file when terminology consistency matters across papers, proposals, chapter drafts, or revision rounds.

## Core rules

- Prefer field-standard technical terms over stylistic paraphrases.
- Once a technically correct term is established in the current draft, keep it stable unless the user requests a change.
- Do not vary key terms merely for stylistic diversity.
- Preserve abbreviations once introduced correctly.
- Keep symbol names, model names, and regime labels unchanged.

## Domain-sensitive vocabulary

Use community-standard wording for areas such as:
- fluidization
- fluidized bed
- circulating fluidized bed
- bubbling fluidized bed
- turbulent fluidization
- gas-solid system
- gas-liquid-solid system
- liquid-solid fluidization
- flow regime
- regime map
- particle clustering
- gas holdup
- entrainment
- drag
- buoyancy
- heat transfer
- mass transfer
- reactor design
- numerical simulation
- CFD
- DEM
- CFD-DEM

## Translation policy

- Translate toward standard English technical usage, not literal Chinese wording.
- If multiple English renderings are possible, choose the one most common in the relevant literature and keep it consistent.
- If the user has an established preferred rendering, keep it unless it is technically misleading.

## Term-check triggers

Provide a short `Term Check` only when one of the following is true:
- the Chinese source is ambiguous
- multiple English renderings are equally plausible
- the user mixes inconsistent terms across sections
- changing one term affects interpretation of the whole passage
