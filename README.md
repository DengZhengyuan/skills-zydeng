## Included Skills

### `writing-master`

Academic English writing skill for:
- fluidized beds
- multiphase flow
- particle technology
- CFD
- DEM
- CFD-DEM
- related modeling and simulation research

It supports:
- Chinese-to-English translation
- English polishing
- mixed Chinese-English draft rewriting
- controlled restructuring
- terminology consistency
- style control based on user-provided exemplars
